В состав системы «Профессор» входит более 60 тестовых методик, проверенных на больших выборках (РФ/СНГ) и показавшие на практике свою эффективность.

В их числе:

Биографические

— BIO БЭМКОН™ (биографический основной) — тест диагностирует социальные качества личности, учитывая эффект компенсации одних качеств другими, по косвенным вопросам (биографические особенности на момент совершеннолетия, то есть на момент рельефного оформления характера человека);
— BS1 БЭМКОН 2 (биографический юношеский).

Характерологические

— KSI опросник Кейрси (темперамент);
— KTL Кеттел: А;
— EKT экспресс-Кеттел;
— XOL характерологический опросник Леонгарда;
— TA1 Айзенка 1 юношеский;
— IZ2 личностный опросник Айзенка (EPI), адаптирован А.Г.Шмелёвым;
— AL1 тест Люшера (8 цветов) для Windows XP и предыдущих;
— MBS Майерс-Бриггс;
— KPM модифицированный опросник Майерс-Бриггс для версии 4.2. и выше.

Интеллектуальные

— IN1 «Количественные отношения»;
— IN2 «Сложные аналогии»;
— IN3 «Аналогии»;
— IN4 «Силлогизмы»;
— IN5 «Общие понятия»;
— INR «Ителлекттест-разминка»;
— IZ1 тест интеллекта Айзенка. IQ тест для версии 4.2. и выше для Windows XP и предыдущих;
— КОТ интеллекттест, КОТ — Краткий Ориентировочный Тест для версии 4.2. и выше.

Мотивационные

— MO2 Мотивация одобрения (Д. Марлоу, Д. Краун) ;
— MO3 Мотивационная установка (Альтруизм — Эгоизм);
— MO4 Мотивационная установка (Труд — Деньги);
— INT Карта интересов;
— MO5 Мотивация избегания неудач Элерса;
— YSP Мотивация успеха Элерса.

Профессиологические

— DDO Дифференциально-диагностический опросник;
— PRS Профессиологический тест для взрослых;
— SSR Способность к коммерческому риску;
— YK1 Управленческие качества 1;
— YK2 Управленческие качества 2 (самооценка);
— YK3 Управленческие качества 3 (бюрократизм);
— YK4 Стиль управления (оценивают руководителя);
— YK5 Стиль управления (оценивает сам руководитель);
— YS1 Управленческие способности 1;
— YS2 Управленческие способности 2;
— YP1 Юношеский профориентационный©.

Углебленное изучение личности

— MPI MMPI Адаптированный MMPI (Minnesota Multiphasic Personality Inventory) 556 вопросов;
— MP2 MMPI сокращенный до основных шкал, 364 вопроса;
— EMP Экспресс-MMPI, сокращённый (очень сильно) вариант теста MMPI, 71 вопрос;
— NPN Нервно-психическая неустойчивость «Прогноз».

Межличностные отношения

— DMO Диагностика межличностных отношений для версии 4.0. и выше;
— LIR Лиритест;
— SC2 Социометрия, количество обследуемых до 33 человек, настройка на группу.

Ценностные ориентации

— SD3, SD4, SOT Семантический дифференциал©, диагностика установок личности на ценности, которые определяет психолог применительно к каждому тестируемому;
— ZE2 Инструментальные ценности Рокича;
— ZEN Ценностные ориентации Рокича, специальные тесты для бесконтактного изучения личности;
— PE1© Диагностика личности по почерку;
— PE2© Диагностика по внешнему виду;
— PE3© Диагностика по поведению.

профессиональные тесты

— OBU© Отбор бухгалтеров;
— OBZ© Отбор начальника службы безопасности;
— ODP© Отбор директора по персоналу;
— OFD© Отбор финансового директора;
— OFO© Отбор начальника финансового отдела;
— OGB© Отбор главного бухгалтера;
— OGD© Отбор генерального директора;
— OKD© Отбор коммерческого директора;
— OKO© Отбор начальника коммерческого отдела;
— OSP© Отбор специалистов.
