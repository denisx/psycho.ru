KPMI (Keys to Personal Mastery Inventory) Ключи персонального мастерства

Первая системная методика развивающей оценки личности, для профессионально-психологической оценки, формирования мотивации на развитие и выявления лидерского потенциала. Разработана специально для использования в бизнес-среде с целью определения индивидуального поведенческого стиля личности.

KPMI помогает выявить сильные стороны и зоны развития сотрудника, прогнозировать его поведение на рабочем месте в различных ситуациях. Результаты опросника могут быть использованы также для прогнозирования ценностных и мотивационных особенностей сотрудника.

Сферы применения

KPMI используется в практике HR для решения следующих бизнес-задач:
— прогнозирование поведения кандидата на рабочем месте;
— формирование эффективных управленческих и проектных команд, налаживание коммуникации в уже существующих командах;
— оценка стиля менеджмента и прогнозирование эффективности деятельности руководителя;
— выявление сопровождение и развитие кадрового резерва;
— профессиональный отбор, построение карьеры.

Функционал

Методика реализована в Интернете, работа с ней не требует установки какого-либо специализированного программного обеспечения на рабочем месте пользователя, за исключением обозревателя Интернета (входит в состав операционной системы).

В результате тестирования формируются развернутые психологические характеристики, прогноз возможных трудностей в профессиональной деятельности и рекомендации по их преодолению.

Руководитель проекта по оценке получает доступ в личный кабинет HR, предоставляющий следующие возможности для управления процессом тестирования и результатами:
— ведение базы данных результатов тестирования;
— сравнение личного профиля каждого сотрудника с «идеальным» профилем успешного руководителя в той или иной сфере;
— сравнение личных профилей сотрудников между собой;
— ранжирование результатов тестирования сотрудников по степени соответствия их личных профилей «идеальному» профилю успешного руководителя в выбранной сфере деятельности;
— ранжирование вероятных должностей для выбранного сотрудника с целью определения наиболее подходящей для него должности;
— формирование «идеального» профиля, профиля команды, профиля должности для своей организации.

Пользователям методики предоставляются телефонные и интернет консультации по вопросам управления Личным кабинетом и работы с результатами тестирования.
